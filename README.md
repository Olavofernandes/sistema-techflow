# Sistema de Gerenciamento de Tarefas - TechFlow Solutions 🚀

Projeto prático desenvolvido para o curso de Análise e Desenvolvimento de Sistemas, aplicando conceitos de Engenharia de Software e Metodologias Ágeis.

## 📋 Escopo do Projeto
CRUD em Python para gerenciamento de tarefas, utilizando armazenamento em memória.
* **Metodologia:** Scrum e Kanban.
* **Design Pattern:** Orientação a Objetos.
* **Testes:** Validação de regras de negócio com PyTest.

## 🛠️ Funcionalidades
- [x] Cadastrar Tarefa
- [x] Editar Tarefa
- [x] Excluir Tarefa
- [x] Listar Tarefas
- [x] Alterar Status (To Do, In Progress, Done)
- [x] Definir Prioridade (Alta, Média, Baixa) - *Mudança de Escopo*

## 💻 Como executar
1. Clone o repositório.
2. Execute o arquivo `main.py` para utilizar o sistema.
3. Para rodar os testes, utilize o comando `python -m pytest`.
